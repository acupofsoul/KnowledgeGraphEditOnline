# 知识图谱可视化快速编辑工具

一个基于React和Ant Design的知识图谱可视化快速编辑工具，支持在画布上直接编辑节点和关系，构建本体，并可接入Neo4j和AI。

## 功能特性

### 核心功能
- **本体构建**：创建、编辑、删除概念和关系类型，支持属性管理
- **图谱编辑**：拖拽创建节点，连接创建关系，支持缩放、平移等画布操作
- **属性编辑**：右侧抽屉式属性编辑表单，支持动态生成属性字段
- **数据管理**：JSON文件导入导出，数据统计展示
- **操作便捷**：右键菜单、键盘操作、双击编辑等多种交互方式

### 技术特点
- **现代化技术栈**：React 18 + TypeScript + Ant Design + Tailwind CSS
- **状态管理**：使用Zustand进行轻量级状态管理
- **图谱可视化**：集成React Flow实现流畅的画布操作
- **响应式设计**：适配不同屏幕尺寸
- **可扩展性**：模块化设计，便于后续集成Neo4j和AI功能

## 快速开始

### 环境要求
- Node.js 16.0+
- npm 7.0+ 或 pnpm 7.0+

### 安装依赖

```bash
# 使用 npm
npm install

# 使用 pnpm
pnpm install
```

### 开发模式

```bash
# 使用 npm
npm run dev

# 使用 pnpm
pnpm dev
```

项目将运行在 http://localhost:5173/（或其他可用端口）

### 构建生产版本

```bash
# 使用 npm
npm run build

# 使用 pnpm
pnpm build
```

构建产物将生成在 `dist` 目录中

## 使用指南

### 1. 本体构建

1. 在左侧导航栏选择「本体构建」
2. 在「概念管理」标签页中创建概念，添加属性
3. 在「关系类型管理」标签页中创建关系类型，添加属性

### 2. 图谱编辑

1. 在左侧导航栏选择「图谱编辑」
2. **基本操作**：
   - 拖拽空白处：平移画布
   - 滚轮滚动：缩放画布
   - 拖拽节点：移动节点位置
   - 双击空白处：创建节点
   - 双击节点：重命名节点
   - 右键点击节点：打开操作菜单
   - 右键拖拽：从源节点到目标节点创建关系
   - 按 Del 键：删除选中元素
   - 按 Esc 键：取消操作
3. **编辑属性**：点击节点或关系，右侧会弹出属性编辑表单

### 3. 数据管理

1. 在左侧导航栏选择「数据管理」
2. **导出数据**：点击「导出为JSON」按钮，将图谱数据导出为JSON文件
3. **导入数据**：拖拽或点击选择JSON文件，导入图谱数据
4. **数据统计**：查看概念、关系类型、节点和关系的数量统计
